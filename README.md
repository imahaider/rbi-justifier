# RBI Justifier

Streamlit app to generate risk justifications for RBI Excel sheets.

## Usage
1. Open the app on Streamlit Cloud.
2. Upload your Excel (must contain required columns).
3. Download the updated Excel with auto-generated justifications.
4. Optionally toggle “LLM Polishing” to improve grammar/flow.

## Required Columns
- Component
- Risk Category
- Driving PoF
- Int Corr Rate
- Ext Corr Rate
- Inspection Priority
- Flamm Conseq Categ
- Toxic Conseq Cat
- Lost Production Category
- Representative Fluid
- Fluid Type
- Initial Fluid Phase
- Toxic Fluid
- Inventory
- Flammable Affected Area
